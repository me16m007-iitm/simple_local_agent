# Local AI Agent

## Overview
This project is designed to implement a local AI agent that can process and analyze realistic restaurant reviews. It utilizes various machine learning techniques to provide insights and recommendations based on the reviews.

## Project Structure
- `main.py`: The main entry point for the application.
- `vector.py`: Contains functions for vectorizing text data.
- `realistic_restaurant_reviews.csv`: A dataset of restaurant reviews used for analysis.
- `requirements.txt`: Lists the dependencies required to run the project.
- `chroma_langchain_db/`: Contains the database files for storing processed data.
- `chroma.sqlite3`: SQLite database file.
- `13652abd-806f-4aa9-9865-78b2154798d1/`: Directory for additional data related to the project.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd local_ai_agent
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```bash
python main.py
```
